# Tableros Etapa 4

Esta carpeta contiene los tableros HTML generados por
`e4_p05_generar_tableros_html`.

## Generación

1. Ejecutar manualmente los scripts `e4_p01` a `e4_p04` desde
   `uv run idw-scripts`.
2. Ejecutar `e4_p05_generar_tableros_html`.
3. Abrir:
   - `tablero_dp01_ventas_geo_score.html`
   - `tablero_dp02_dqm.html`

Los HTML generados son artefactos de publicación: se pueden entregar tal cual o
usar como guía para construir un PBIX sobre las tablas `DPxx_`.
